# О проекте
Экспериментальный проект, интересно было разбить на многоуровневость классического сапёра и посмотреть что из этого выйдет. "Искусственно" симулировано 10 фпс, ради таймера. Это не полноценная релизная игра, а лишь тестовый кусочек. Может в дальнейшем вернусь к этой мысли.

# Публикация framework-dependent (нужен установленный .NET Runtime)
dotnet publish -c Release -o publish/

# Запускаем EXE
publish/Miner.exe
