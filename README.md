# О проекте
Мысленный экспериментальный проект, превратившийся в мини-игру про сапера. Это не полноценная релизная игра, а лишь не совершенные наброски. В проекте присутствуют проблемы, он сложно мастшабируем и требуется более глубокий рефакторинг.

# Как запустить и сформировать рабочий EXE
Пример на Windows CMD / PowerShell.

REM 1. Клонировать репозиторий
git clone Miner

REM 2. Перейти в директорию проекта
cd <ваш путь>\Miner

REM 3. Собрать проект в режиме Release
# Публикация framework-dependent (нужен установленный .NET Runtime)
dotnet publish -c Release -o ./publish

REM 4. Перейти в директорию с собранным EXE
cd publish

REM 5. Запустить консольное приложение с аргументами
publish/Miner.exe
